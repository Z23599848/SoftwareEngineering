\section{User Stories}

\begin{itemize}
    \item \textbf{US1 – Create profile:} Create a profile with course, year, and modules.
    \item \textbf{US2 – Login to account:} Secure profile and activity.
    \item \textbf{US3 – Search by module:} Find someone studying the same subject.
    \item \textbf{US4 – View user profiles:} Decide if a student is a good match.
    \item \textbf{US5 – Send study request:} Organise a study session.
    \item \textbf{US6 – Accept or decline requests:} Control who you study with.
    \item \textbf{US7 – Dashboard:} See upcoming sessions and pending requests.
    \item \textbf{US8 – Filter by availability:} Find someone free at similar times.
    \item \textbf{US9 – Ratings and feedback:} Leave feedback after a session.
    \item \textbf{US10 – Report or block a user:} Safety against inappropriate behavior.
\end{itemize}
